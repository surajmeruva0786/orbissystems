Orbis Systems proposes to design, develop, and deploy a comprehensive AI-powered Zoo & Wildlife Sanctuary Management System for Nandanvan Jungle Safari, Nava Raipur. The system replaces manual paper-based records with a modern digital platform available on web, Android mobile, and desktop — with voice-enabled reporting in Hindi and Chhattisgarhi, AI-assisted health monitoring, and data migration from existing paper records. All deliverables listed in this document are included in the agreed project scope with 1 year of post-launch support.

Complete List of Deliverables
D-01 Web Application Core Platform
  • Fully responsive web application accessible from any modern browser
  • Secure login with role-based dashboard routing for all 4 user types
  • Bilingual interface — English and Hindi
  • Role-specific access control — each user sees only their permitted sections
D-02 Desktop Application (Windows) Desktop
  • Standalone Windows desktop application for office-based staff
  • Full access to all role-based features, optimized for desktop workflows
  • Installable without browser dependency — runs directly on Windows PCs
  • Synchronized in real-time with the web and mobile platforms
D-03 Android Mobile Application (Field-Ready) Mobile
  • Installable Android app — no Play Store required
  • Voice recording directly in the field for log submissions
  • Touch-optimized interface for outdoor / sunlight conditions
  • Push notifications for log deadlines and emergency SOS alerts
  • Camera integration for attaching photos to animal health reports
D-04 Role-Based Dashboards — 4 Distinct User Interfaces Core Platform
Zookeeper
  • Submit AM & PM animal observation logs (feeding, health, behavior)
  • Voice-to-text log entry in Hindi / Chhattisgarhi with AI auto-fill
  • Photo attachments for veterinary review
  • Push reminders at 11:00 AM and 4:00 PM for pending logs
  • Emergency SOS button — instant broadcast to all staff
Veterinary Doctor
  • Complete medical history and treatment records per animal
  • Medication prescription and dosage tracking
  • AI-escalated health alert queue for urgent cases
Forest Officer
  • Feeding cost analytics and species-wise expenditure overview
  • Inventory tracking for food supplies with low-stock alerts
  • One-click CSV and PDF export for official reports
Administrator
  • System-wide overview — animal count, health alerts, log completion rates
  • Staff account management — add, edit, assign roles
  • Animal profile management — species, enclosure, health history
D-05 AI & Voice Features AI / ML
  • Voice log entry in Hindi and Chhattisgarhi with automatic transcription
  • AI-powered auto-fill of structured log forms from voice input
  • Automatic health keyword detection for veterinary escalation
  • Multilingual translation to English for standardized record storage
  • Bilingual UI toggle — switch between English and Hindi in real-time
D-06 Database, Storage & Cloud Infrastructure Infrastructure
  • Real-time cloud database for all animal, health, staff, and inventory records
  • Secure media storage for photos and voice recordings (per-animal)
  • Role-based data access — each user can only access their permitted data
  • Push notification infrastructure for reminders and SOS alerts
  • Automated daily data backups for disaster recovery and audit compliance
  • Server-side PDF and CSV report generation
D-07 Deployment, Hosting & Custom Domain Deployment
  • Production deployment of web app and backend with 24/7 uptime
  • Custom domain setup with SSL certificate (e.g., junglesafari.cg.gov.in or similar)
  • Secure management of all credentials and API keys
  • Automated deployment pipeline for reliable updates
  • Desktop application build and installer package delivery
D-08 Data Migration from Paper Records Migration
  • Migration of existing animal profiles and records from paper-based logs into the digital system
  • Digitization and structured entry of historical health and treatment records
  • Verification and quality check of migrated data before go-live
D-09 Documentation, Training & 1-Year Support Support
  • Role-specific user manuals in English and Hindi
  • Live onboarding and training session for all staff categories
  • Full source code handover upon project completion and final payment
  • 1-year post-launch bug-fix and support coverage at no additional charge
